# Murali Varma - Personal Website

This is the personal website of Murali Varma, built with [Astro](https://astro.build).

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Migration Notes

This site was migrated from VuePress to Astro. The migration process involved:

1. Setting up a basic Astro project
2. Configuring Vue integration (for compatibility with existing Vue components)
3. Migrating content from Markdown to Astro components
4. Preserving the design and functionality of the original site

## Technologies Used

- **Astro**: Main framework
- **Vue**: For components migrated from VuePress
- **Markdown**: For content

## ğŸ§ Commands

All commands are run from the root of the project, from a terminal:

| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| `npm install`             | Installs dependencies                            |
| `npm run dev`             | Starts local dev server at `localhost:4321`      |
| `npm run build`           | Build your production site to `./dist/`          |
| `npm run preview`         | Preview your build locally, before deploying     |
| `npm run astro ...`       | Run CLI commands like `astro add`, `astro check` |
| `npm run astro -- --help` | Get help using the Astro CLI                     |
