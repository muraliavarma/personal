Complete 3D rendering pipeline implemented in a single HTML page — fully interactive in the browser.

- Covers world transforms, lighting, view/perspective transforms, z-clipping, culling, z-sorting, and rasterization
- Uses Sylvester for matrix math and Raphael for rasterization
- Users can adjust object and light positions in 3D space with immediate visual feedback
