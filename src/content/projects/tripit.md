Led development of TripIt's new web app, built from scratch in Vue.js and integrated with existing AWS ECS infrastructure and CI/CD pipeline.

- Designed auth architecture: cookie-based over OAuth2 to minimize attack vectors
- Built serializer/deserializer framework abstracting legacy API for future migration
- Launched EU rollout — reworked cross-region session management from US-only assumptions
- Conducted architectural reviews with threat modeling team; mentored junior developers
- Features shipped: COVID Guidance, email management, localization, inbox sync
