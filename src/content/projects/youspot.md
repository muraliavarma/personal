Chrome extension that parses YouTube song mixes and adds selected tracks to your Spotify playlists.

- Built with AngularJS and Node.js
- Handles Spotify OAuth integration and Chrome Extension CORS challenges
- Users select matching tracks (green/grey) and add them to any existing or new Spotify playlist
