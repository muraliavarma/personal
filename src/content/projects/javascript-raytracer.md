Interactive raytracer running in the browser with real-time 3D manipulation.

- Full rendering pipeline: world transforms, lighting, perspective projection, z-clipping, culling, z-sorting, rasterization
- Adjust object and light positions in 3D space with immediate visual feedback
- Uses Sylvester for matrix calculations and Raphael for rasterization
