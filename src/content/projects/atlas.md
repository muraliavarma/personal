Full-stack geospatial data visualization platform presenting complex data as interactive tables and maps.

- Vue.js frontend, Flask/Python backend, MongoDB, deployed on Kubernetes
- Built custom open-source table component with sticky columns and grouped headers (Vue scoped slots)
- Implemented full Keycloak auth stack: token exchange, JS library integration, login theming via K8s init containers with Helm and Apache Freemarker
